<task type="auto">

  <files>

  <action>
    **Step 1: Create Embind spike** -- `spike-embind/main.cpp` using the Embind pattern from RESEARCH.md:
    ```cpp
    #include <emscripten/bind.h>
    #include <string>

    std::string greet(const std::string& name) {
        return "Hello from WASM, " + name + "!";
    }

    EMSCRIPTEN_BINDINGS(spike_embind) {
        emscripten::function("greet", &greet);
    }
    ```

    `spike-embind/CMakeLists.txt`:
    ```cmake
    add_executable(spike-embind main.cpp)
    target_link_options(spike-embind PRIVATE -lembind)
    set_target_properties(spike-embind PROPERTIES
        SUFFIX ".js"
        LINK_FLAGS "-sMODULARIZE=1 -sEXPORT_ES6=0 -sENVIRONMENT=node"
    )
    ```
    Note: Use `-lembind` (modern), not `--bind` (deprecated). Use `-sMODULARIZE=1` so the output is importable as a module (per Pitfall 5 in research).

    **Step 2: Create Node.js test** at `test/test-embind.mjs`:
    ```javascript
    import { createRequire } from 'module';
    const require = createRequire(import.meta.url);
    const createModule = require('../build/spike-embind/spike-embind.js');
    const Module = await createModule();
    const result = Module.greet("NebulaStream");
    console.assert(result === "Hello from WASM, NebulaStream!", `Unexpected: ${result}`);
    console.log("PASS: Embind spike");
    ```
    Note: Use `createRequire` because MODULARIZE with EXPORT_ES6=0 produces CommonJS. If import fails, try dynamic `import()` or adjust EXPORT_ES6 flag.

    **Step 3: Build and test**:
    ```bash
    cd nes-topology-editor/wasm
    source emsdk/emsdk_env.sh
    cmake --build build --target spike-embind
    node test/test-embind.mjs
    ```

    **Step 4: Update top-level CMakeLists.txt** to include spike-embind subdirectory if not already added in Task 1.
