  <action>
    Present the following verification checklist to the user.

    What was built: Complete Phase 4 polish features -- undo/redo, keyboard shortcuts, topology validation overlays, ANTLR SQL validation, sidebar toggle, and dark mode.

    How to verify:
    1. Start the dev server: `cd nes-topology-editor && npm run dev`
    2. Open http://localhost:5173

    **Undo/Redo (UIPL-01):**
    3. Add a worker node via toolbar. Press Ctrl+Z -- the worker should disappear (undone). Press Ctrl+Shift+Z -- the worker should reappear (redone).
    4. Add a source, attach it to a worker, then Ctrl+Z twice -- source should detach, then disappear.
    5. Open the YAML overlay, click inside the Monaco editor, press Ctrl+Z -- Monaco should undo its own text change (NOT the store). Close overlay, press Ctrl+Z -- store undo should work.

    **Keyboard Shortcuts (UIPL-02):**
    6. Select a node. Press Delete -- node should be removed.
    7. Open YAML overlay. Press Escape -- overlay should close.
    8. Click a node (sidebar opens). Press Escape -- node should deselect and sidebar should clear.

    **Topology Validation (PROP-06):**
    9. Add a source but do NOT attach it to a worker. Verify a warning icon (orange triangle) appears on the source node.
    10. Hover the warning icon -- tooltip should say "Source not attached to any worker" or similar.
    11. Attach the source to a worker -- warning icon should disappear.
    12. Verify YAML export still works even with validation warnings present.

    **SQL Validation (QURY-04):**
    13. Open the query panel, add a query.
    14. Type `SELECT * FROM source1` -- no red underlines should appear.
    15. Type `SELEC * FROM source1` (typo) -- red squiggly underline should appear after ~500ms.
    16. Fix the typo -- red underline should disappear.

    **Dark Mode (UIPL-04):**
    17. Click the sun/moon toggle in the toolbar. Entire app should switch to dark mode (dark backgrounds, light text, dark canvas).
    18. Refresh the page -- dark mode should persist (no flash to light then dark).
    19. Toggle back to light mode. Refresh -- should stay light.

    **Sidebar Toggle (UIPL-03):**
    20. Click a node -- sidebar should show properties. Press Escape -- sidebar content should clear.

    Resume signal: Type "approved" if all features work correctly, or describe any issues found.
