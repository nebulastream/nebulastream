  <action>
    1. **Install Monaco**: Run `cd nes-topology-editor && npm install @monaco-editor/react`

    2. **QueryPanel.tsx**: Create the bottom panel component:
       - **Resizable top edge**: Same pattern as sidebar resize but vertical. Min height 120px, max 500px, default 250px. 4px resize handle on top edge with cursor-row-resize.
       - **Collapse/expand**: Toggle button (lucide ChevronDown/ChevronUp) that sets height to 36px (just showing the tab bar) or restores previous height.
       - **Tab bar** (horizontal, scrollable):
         - Each query rendered as a tab: shows `query.name || '(unnamed)'` in italic if unnamed
         - Active tab has blue bottom border + bg-white, inactive has bg-gray-50
         - Click tab: `selectQuery(query.id)`
         - Double-click tab: switch to inline name editing mode (input replaces text). On blur/Enter, save name via `updateQuery(id, { name })`. Empty name is allowed (shows unnamed).
         - Close button (X) on each tab: `removeQuery(query.id)`. If removing active query, select the next one or null.
         - "+" button at end of tabs: `addQuery()` then `selectQuery(newId)` -- to get the new id, read queries after addQuery and pick the last one
       - **Editor area**:
         - If `selectedQueryId` is set and a matching query exists: render Monaco `<Editor>` with:
           - `defaultLanguage="sql"`
           - `value={selectedQuery.sql}`
           - `onChange={(v) => updateQuery(selectedQueryId, { sql: v ?? '' })}`
           - `options={{ minimap: { enabled: false }, lineNumbers: 'on', fontSize: 13, scrollBeyondLastLine: false, wordWrap: 'on', automaticLayout: true }}`
           - `loading={<div className="p-4 text-gray-400">Loading editor...</div>}`
           - `height="100%"` with parent div having explicit calculated height (panel height minus tab bar height ~36px minus resize handle ~4px)
         - If no query selected or no queries exist: empty state "Add a query to get started" with + button

    3. **QueryPanel.test.tsx**:
       - Mock Monaco editor: `vi.mock('@monaco-editor/react', () => ({ default: (props: any) => <textarea data-testid="monaco" value={props.value} onChange={(e) => props.onChange?.(e.target.value)} /> }))` -- replaces Monaco with a textarea for testing
       - Mock zustand store with vi.hoisted() pattern
       - Test: renders "Add a query" empty state when no queries
       - Test: renders query tabs with names
       - Test: unnamed query shows "(unnamed)" text
       - Test: clicking "+" calls addQuery
       - Test: clicking "X" calls removeQuery
