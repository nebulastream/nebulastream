    - copySelectedNodes serializes selected nodes and edges between them, excludes edges to unselected nodes

       - Edge count preserved after layout
       - Empty graph returns empty

       - `getLayoutedElements(nodes: Node[], edges: Edge[], direction = 'TB'): { nodes: Node[], edges: Edge[] }`

         - Returns serializable { nodes, edges }
       - `pasteNodes(clipboard: ClipboardData, offset = {x: 50, y: 50}): { nodes: Node[], edges: Edge[] }`
         - Generates new IDs for each node (mapping old->new). Updates positions with offset.
         - Generates new edge IDs, remaps source/target using the ID mapping.

    5. Run all tests.
