       - Use `useStore(selectReactFlowNodes)` and `useStore(selectReactFlowEdges)` for nodes/edges

         - Use `screenToFlowPosition({x: event.clientX, y: event.clientY})` to convert coordinates
         - Call store.addWorker(position) for 'worker' type
         - Call store.addPhysicalSource({...defaults, position}) for 'source' type
         - Call store.addSink({...defaults, position}) for 'sink' type
       - Props on ReactFlow: snapToGrid, snapGrid={[20, 20]}, deleteKeyCode="Backspace", fitView
       - Children: Background (variant Dots, gap 20), MiniMap (zoomable, pannable), Controls
       - Note: Connection handling (onConnect, isValidConnection) will be wired in Plan 04

       - Style: vertical sidebar on the left, items styled like cards with icons matching their node type
